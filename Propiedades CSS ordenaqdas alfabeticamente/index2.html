  • align-content:

    Establece la alineación vertical del contenido de un contenedor flexible cuando hay espacio disponible en el eje transversal.

  • align-items:

    Establece la alineación vertical de los elementos dentro de un contenedor flexible.

    Establece todas las propiedades CSS a sus valores iniciales o heredados.

  • animation:

    Una propiedad abreviada para establecer varias propiedades de animación a la vez.

  • animation-delay:

    Establece un retraso antes de que comience una animación.

  • animation-direction:

    Establece si una animación debe reproducirse hacia adelante, hacia atrás o en ambas direcciones.

  • animation-duration:

    Establece la duración de una animación.

  • animation-fill-mode:

    Establece qué valores deben aplicarse a un elemento antes y después de una animación.

  • animation-iteration-count:

    Establece el número de veces que se debe reproducir una animación.

  • animation-name:

    Establece el nombre de una animación definida por el usuario.

  • animation-play-state:

    Establece si una animación está en pausa o en reproducción.

  • animation-timing-function:

    Establece la curva de tiempo de una animación.

  • backface-visibility:

  • background-size:
