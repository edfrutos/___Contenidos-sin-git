EDF - Propiedades CSS

  • align-content: Establece la alineación vertical del contenido de un contenedor flexible cuando hay espacio disponible en el eje transversal.
  • align-items: Establece la alineación vertical de los elementos dentro de un contenedor flexible.
  • align-self: Establece la alineación vertical de un elemento en relación con su contenedor flexible.
  • all: Establece todas las propiedades CSS a sus valores iniciales o heredados.
  • animation: Es una propiedad abreviada que permite definir una animación en CSS.
  • background: Establece el fondo de un elemento.
  • border: Establece el ancho, el estilo y el color del borde de un elemento.
  • box-shadow: Establece una sombra alrededor de un elemento.
  • color: Establece el color del texto de un elemento.
  • display: Establece cómo se muestra un elemento en el documento.
  • font-family: Establece la familia de fuentes utilizada para el texto de un elemento.
  • font-size: Establece el tamaño de letra utilizado para el texto de un elemento.
  • font-style: Establece el estilo de la fuente utilizada para el texto de un elemento.
  • font-weight: Establece el grosor de la fuente utilizada para el texto de un elemento.
  • height: Establece la altura de un elemento.
  • margin: Establece el margen exterior de un elemento.
  • padding: Establece el relleno interior de un elemento.
  • text-align: Establece la alineación horizontal del texto de un elemento.
  • width: Establece el ancho de un elemento.
